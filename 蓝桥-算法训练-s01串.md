#               蓝桥-算法训练-s01串

描述：s01串初始为"0"按以下方式变换，0变1，1变01

解释一下啥是s01串：就是f(n)=f(n-2)+f(n-1)

输入格式:
　　1个整数(0~19)
输出格式:
　　n次变换后s01串

样例：输入3，输出101

样例解释：0-1

​                   1-01

​                   01-101 即结果是101

思路：直接模拟？很奇怪唉，因为很多代码上面就直接模拟了整个过程，代码看起来很简略

上代码：

```
#include <iostream>
using namespace std;
string fun(int n);
int main()
{
    int n;
    cin >> n;
    cout << fun(n) << endl;
    return 0;
}
string fun(int n)
{
    if(n == 0)
        return "0";
    else if(n == 1)
        return "1";
    else
        return fun(n-2) + fun(n-1);
}


```
